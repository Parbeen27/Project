# Portfolio of Parbeen Singh Panwar

## Overview

This portfolio website showcases the work and skills of Parbeen Singh Panwar, a BCA graduate, web developer, and freelancer. The site is built with HTML, CSS, and JavaScript, and includes animated sections, responsive layout, and a modern presentation of services, portfolio work, and contact details.

## Live Page View
- https://parbeen27.github.io/Project/

## Project Structure

- `index.html` - Main portfolio website entry page.
- `css/` - Primary stylesheet files, including responsive design rules.
- `JavaScript/` - JavaScript files used for interactive page behavior and animations.
- `images/` - Image assets used across the website.
- `.gitignore` - Git ignore rules for the project.

## Features

- Hero section with animated typing effect
- Responsive navigation menu
- Portfolio showcase with project visuals
- Services section outlining web development offerings
- Contact call-to-action for hiring and inquiries
- AOS animation library integration for smooth scroll effects

## How to Use

1. Open `index.html` in a web browser.
2. Use the top navigation to jump between sections like Home, About, Portfolio, Services, and Contact.
3. Review project samples and reach out using the call-to-action buttons.

## Development Notes

- Update `css/style.css` and `css/responsive.css` to change the visual design.
- Modify `JavaScript/` files to change interactive behavior or section scrolling.
- Replace image files in `images/` to showcase new portfolio assets.

## Deployment

This is a static website and can be hosted on platforms like GitHub Pages, Netlify, or Vercel. Simply deploy the repository root and ensure `index.html` is served as the landing page.
